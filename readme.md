## Tech Stack:
- JavaScript
- Express.JS
- Socket.IO
- Deployed using Heroku

## App Functionality
- Websocket implementation
- Write to chat and different chat rooms
- View real time chat and different chat rooms
- Setting username
- Visit different chat rooms on choice
